# learn-map

## 在vue项目中使用高德API

### 一、获取高德 key

### 二、在public下的模板文件index.html中加入     
### <script src="https://webapi.amap.com/maps?v=1.4.15&key=your key"></script>

### 三、在使用高德API页面style标签中导入样式
### @import url("https://a.amap.com/jsapi_demos/static/demo-center/css/demo-center.css");

### 到此处就可以正常使用了
